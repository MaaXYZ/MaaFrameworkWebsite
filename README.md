# MaaFramework 文档网站

> [!TIP]
> 本仓库不包含文档文件。如果你希望编写/修正文档，请前往 [MaaFramework 主仓库](https://github.com/MaaXYZ/MaaFramework/tree/main/docs)

## 完善站点

### 开发

1. clone 此项目至本地

```shell
git clone https://github.com/MaaXYZ/MaaFrameworkDocsWebsite
```

2. 装载依赖

```shell
cd ./MaaFrameworkDocsWebsite
yarn
```

3. 启动调试

```shell
yarn dev
```

### 打包

```shell
yarn build
```
